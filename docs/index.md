# Vintage Story Mod Loader

Welcome to the Vintage Story Mod Loader documentation!

## Overview

The Vintage Story Mod Loader is a cross-platform desktop application that makes it easy to manage mods and mod packs for Vintage Story. It provides a modern, user-friendly interface for:

- **Browsing and downloading mods** from the Vintage Story mod database
- **Managing installed mods** with batch enable/disable operations
- **Creating and sharing mod packs** with other players
- **Editing mod configurations** directly from the application
- **Submitting mod packs** to the mod database

## Quick Start

### For Users

1. Download and install the Mod Loader for your platform
2. Launch the application
3. The Mod Loader will automatically detect your Vintage Story installation
4. Browse mods, download them, and manage your mod collection

See the [User Guide](/user-guide) for detailed instructions.

### For Mod Pack Designers

1. Create a mod pack by selecting mods from your installed collection
2. Add metadata, descriptions, and screenshots
3. Export your mod pack or submit it directly to the mod database

See the [Designer Guide](/designer-guide) for detailed instructions.

## Features

- **Cross-platform**: Works on Windows, macOS, and Linux
- **Batch Operations**: Enable or disable multiple mods at once
- **Mod Pack Support**: Create, import, and apply mod packs
- **Configuration Editor**: Edit mod config files with syntax highlighting
- **Auto-detection**: Automatically finds your Vintage Story installation
- **Integration**: Seamlessly integrates with the Vintage Story mod database
